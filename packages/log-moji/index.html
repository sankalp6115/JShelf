Log-Moji

"Logging with style, clarity, and a touch of emoji magic."
A zero-dependency JavaScript logging utility that enhances your browser console output with colored labels, emoji icons, timestamps, and optional stack traces.

Installation & Usage

import { log } from 'https://cdn.jsdelivr.net/gh/sankalp6115/jshelf/packages/log-moji/log-moji.js';

Stop-Plague JS

A lightweight JavaScript shield to block right-clicks, hotkeys, and casual content theft in seconds.

Installation & Usage

import * as stopPlague from 'https://cdn.jsdelivr.net/gh/sankalp6115/stopPlague@latest/stopPlague.js';

import { stopCtrlKey } from 'https://cdn.jsdelivr.net/gh/sankalp6115/stopPlague@latest/stopPlague.js';

stopPlague.stopCtrlKey(['a','s']);

Get Started

Function Description Example 
stopCtrlKey(keys = [])
 Blocks any Ctrl + [key] combinations where [key] matches an item in the provided array. Useful for preventing shortcuts like Ctrl+U (View Source) or Ctrl+C (Copy). 
stopCtrlKey(['u', 'c', 'p']) 
// Blocks Ctrl+U, Ctrl+C, Ctrl+P
 
stopCtrlShiftKey(keys = [])
 Blocks any Ctrl + Shift + [key] combinations where [key] is in the provided array. Often used to block developer tool shortcuts like Ctrl+Shift+I or Ctrl+Shift+J. 
stopCtrlShiftKey(['i', 'j']) 
// Blocks Ctrl+Shift+I, Ctrl+Shift+J
 
stopCopy(replaceText = "")
 Prevents copying text via Ctrl+C. If replaceText is provided, that text will be placed into the clipboard instead. If not provided, a default warning message will be copied. 
stopCopy("Copying is disabled here!") 
// Replaces copied text with the given string
 
stopRightClick(showMessage = false)
 Disables the right-click context menu. If showMessage is true, an alert will be shown when the user right-clicks. 
stopRightClick(true) 
// Disables right-click and shows an alert
 
stopF12()
 Blocks pressing the F12 key, which usually opens browser developer tools. 
stopF12() 
// Disables F12 DevTools shortcut
 
disableTextSelect()
 Disables text selection for all elements on the page, except inputs, textareas, buttons, and forms to preserve normal interaction. 
disableTextSelect() 
// Prevents selecting page text

Examples

        initContentGuard({
          ctrlKeys: ['u', 'c', 'p'],        // Blocks Ctrl+U, Ctrl+C, Ctrl+P
          ctrlShiftKeys: ['i', 'j'],        // Blocks Ctrl+Shift+I, Ctrl+Shift+J
          replaceCopyText: "Nice try!",     // Replaces copied content with custom text
          blockRightClick: true,            // Disables right-click
          showRightClickMsg: false,          // Shows alert when right-click blocked
          blockF12: true,                   // Blocks F12 DevTools
          disableTextSelect: true           // Disables text selection
        });
OR
        sp.stopCtrlKeys(['u', 'c', 'p']);     // Blocks Ctrl+U, Ctrl+C, Ctrl+P
        sp.stopCtrlShiftKeys(['i', 'j']);     // Blocks Ctrl+Shift+I, Ctrl+Shift+J
        sp.stopCopy("Don't Copy");            // Prevents user from copying content
        sp.stopRightClick(false),             // Disables right-click
        sp.stopF12();                         // Blocks F12 DevTools
        sp.disableTextSelect();               // Disables text selection

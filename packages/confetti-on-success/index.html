Confetti On Success

A lightweight JavaScript utility that celebrates success events with beautiful confetti animations, emoji particles. Perfect for rewarding users after completing tasks, achieving milestones, or unlocking Easter eggs.

Installation & Usage

import confettiOnSuccess from 'https://cdn.jsdelivr.net/gh/sankalp6115/JShelf/packages/confetti-on-success/confettiOnSuccess.js';

Get Started

Options Description Example 
theme
 Choose from party, elegant, crazy, easterEgg confettiOnSuccess({ theme: 'crazy' }); (default: randomly from (party,elegant,crazy and secret easter egg)) 
particleCount
 Number of confetti particles confettiOnSuccess({particleCount: 200}); (default: 100) 
colors
 Array of hex color codes for confetti confettiOnSuccess({ colors: ['#ff0000', '#00ff00', '#0000ff'] }); 
duration
 Animation duration confettiOnSuccess({ duration: 3000 });

 Floating success banner message confettiOnSuccess({ message: "ðŸŽ‰ You did it!" });
